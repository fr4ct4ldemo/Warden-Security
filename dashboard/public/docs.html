Privacy Policy

Last updated: May 25, 2026

Our Commitment

Warden is committed to protecting your privacy. We collect only the minimum data necessary to provide the service and never sell or share your data with third parties.

1. Data We Collect

Server Data

  • Guild ID and configuration settings
  • Moderation cases (bans, kicks, warns, mutes) including user IDs, moderator IDs, reasons, and timestamps
  • Automod settings and ignored channels
  • Verification settings and role assignments
  • Warn punishment thresholds

User Data

  • Discord User ID (for moderation records and warn history)
  • Username at time of action (for log readability)
  • OAuth2 access token (temporarily for dashboard login — never stored permanently)

2. How We Use Data

  • To provide moderation features (bans, mutes, warns)
  • To enforce automod rules configured by server admins
  • To display moderation history and case logs
  • To authenticate dashboard users via Discord OAuth2

3. Data Storage

All data is stored locally in a SQLite database on the server running Warden. Data is not transmitted to any external service or cloud provider.

4. Data Retention

Server data is retained as long as Warden remains in the server. You may request deletion of your server's data by contacting us through the support server.

5. Third-Party Services

Warden interacts with Discord's API as required by its functionality. Discord's own privacy policy governs data processed by Discord. Links are checked against public phishing databases — no personal data is sent in these requests.

6. Your Rights

  • You may request a copy of all data stored for your server
  • You may request deletion of your server's data at any time
  • Server administrators control what data Warden collects by configuring its features

7. Changes to This Policy

We may update this Privacy Policy from time to time and will notify users of significant changes via our support server.

8. Contact

For privacy-related requests, join our Support Server.
